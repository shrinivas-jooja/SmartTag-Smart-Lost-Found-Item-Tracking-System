# SmartTag – Smart Lost & Found Item Tracking System

## 📌 Overview

SmartTag is a QR code-based Lost & Found platform designed to help users recover misplaced belongings quickly and securely. Each item is assigned a unique SmartTag (QR code). When someone finds a lost item, they simply scan the QR code to notify the owner without revealing the owner's personal information.

The system focuses on privacy, ease of use, and fast communication between the finder and the owner.

---

## ✨ Features

- 🔖 Generate a unique QR code for every registered item
- 👤 Secure user registration and authentication
- 📦 Register and manage multiple personal items
- 📱 Scan SmartTag to report a found item
- 🔒 Protect owner's personal information
- 📩 Notify the owner when an item is found
- 🌐 Simple and responsive user interface

---

## 🚀 How It Works

1. User creates an account.
2. Register an item in the system.
3. A unique SmartTag (QR Code) is generated.
4. Attach the SmartTag to the item.
5. If the item is lost and someone finds it, they scan the QR code.
6. The finder is redirected to a secure webpage.
7. The owner receives a notification and can communicate securely with the finder.
8. The item is successfully recovered.
